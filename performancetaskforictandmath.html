The domain of a function or relation is the set of all possible independent values the relation can take.

The range of a function or relation is the set of all possible dependent values the relation can produce from the domain values.

4. Dependent & Independent

A painter must measure a room before deciding how much paint to buy.

Dependent variable:

How much or amount of paint to buy.
Independent variable:

Measurement of the room.

An independent variable is a variable that represents a quantity that is being manipulated in an experiment. A dependent variable represents a quantity whose value depends on those manipulations. The independent variable belongs on the x-axis (horizontal line) of the graph and the dependent variable belongs on the y-axis (vertical line).
